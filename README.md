# INSTACLONE

Instaclone – Full Stack Social Media Application

Tech: Node.js, Express.js, MongoDB, React, JWT, Axios, ImageKit, Render

Built a full-stack social media application supporting user authentication, post creation, likes, follow/unfollow, and follower request management.

Designed a graph-style relationship system using edge collections in MongoDB to manage followers, following, and pending requests efficiently.

Implemented secure authentication with JWT stored in HTTP-only cookies, including login, registration, session persistence, and logout.

Developed RESTful APIs with Express and Mongoose, including features like feed generation, suggested users, and follower/following lists.

Optimized database queries using compound indexing and population of referenced documents for faster relationship queries.

Integrated ImageKit cloud storage for handling post images and user profile pictures.

Architected a React frontend using a 4-layer structure (UI → Hooks → State → API) for scalable state and API management.

Deployed the application on Render, serving both backend APIs and the built React frontend from the same server.
